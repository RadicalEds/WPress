# WPress
Installs and Configures a LAMP Server with a Wordpress VHost.

### Requirements
- An Internet Connection
- Debian with Apt
- root access for installing
- A Domain Name already configured with the servers address

Any missing packages will be installed by apt    

### Features
- Install/Configure LAMP Server with all wordpress features
- Enables TLS with LetsEncrypt
- Creates a new MYSQL User/Database (if needed)
- Installs and Automatically Configures Wordpress
- URL Rewriting and Permalinks will work without any fuss

### Planned Features
- Security Tweaks that Every Wordpress Should Have by Default
- Auto Install/Configure Plugins/Themes
- The Ability to Modify a Current Installation, Import/Export Pages/Posts/Products, Install/Update Plugins/Themes etc.
